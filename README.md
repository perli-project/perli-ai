## ✨ AI 주요 기능
> AI 스마트 카드 실적 매니저의 AI 엔진은 마이데이터로 수집된 카드 사용 내역을 분석하여 <br>
사용자별 소비 패턴을 학습하고, 전월 실적 달성 가능성 예측,<br>
카드별 혜택 최적화 전략, 지출 분배 추천을 자동 생성하는 지능형 모델입니다.<br>
규칙 기반과 머신러닝 기반을 결합한 하이브리드 AI 구조로 높은 정확성과 개인화된 추천을 제공합니다.
